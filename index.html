WEBAUDIOVISUALIZER

Easy to set up HTML web audio api visualizers.

DOWNLOAD

Grab the full script, or the minified version.

For now, you'll also need jQuery, so yeah, there's that

SET UP

Add the webAudioVisualizer.js file to your project, create a webAudioVisualizer object and a div to hold it, and connect it to your node.

We'll use a superKeyborad to generate the sound


      // In Script file

      $(document).ready(function(){ 
        let keyborad = new Keyborad()
        let c = keyborad.context
        let n = keyborad.mixNode

        //Create the visualizers, give them a context, container and type
        //the full options list is shown, but params will fall back on default
        // values if not provided
        let audioVisualizerVolume = new AudioVisualizer({context:c,
                        divID:'volumeID',
                        type:'Volume', //type can be Volume/Frequency/Oscilloscope
                        width:400,
                        height:220,
                        fftSize:2048})
                        
        //Connect to the desired node
        audioVisualizerVolume.connectAndDraw(n)

        let audioVisualizerFreq = new AudioVisualizer({context:c,divID:'freqID',type:'Frequency'})
        audioVisualizerFreq.connectAndDraw(n)

        let audioVisualizerScope= new AudioVisualizer({context:c,divID:'scopeID',type:'Oscilloscope'})
        audioVisualizerScope.connectAndDraw(n)
      })

      // In HTML file

      <div id="keyborad"></div>;

      <div id="volumeID"></div>
      <div id="freqID"></div>
      <div id="scopeID"></div>

And that's it
